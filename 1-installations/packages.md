# Dépendances

---

## Front-End

### Babel

- `npm i -D babel/core babel/preset-en`

### Standard

- `npm i -D standard`

Dans les settings de VSCODE `settings.json` ajouter les lignes suivantes :

`"standard.enable": true,
"standard.run": "onType",
"standard.autoFixOnSave": true
`
---

## Back-End

### Bcrypt js

- `npm i bcryptjs`

### Json Web Token

- `npm i jsonwebtoken`

### Base de données

- `npm i sqlite3`
